## 本轮修复的 Bug

| # | 问题 | 根因 | 修复 |
|---|------|------|------|
| 1 | 每日建议 500 错误 | `suggestion.user_id` 无默认值 | `createMockSuggestion` 增加 userId 参数 |
| 2 | 每日建议 500 错误 | `suggestion_id1` 列名不匹配 | 添加 `@TableField("suggestion_id_1")` 注解 |
| 3 | 每日建议 500 错误 | date 参数为 null 时未默认当天 | 添加 `LocalDate.now()` 默认值 |
| 4 | 成就列表为空 | 无种子数据 | 插入 6 条成就初始数据 |
| 5 | 错误消息不准确 | ExceptionHandler 用默认消息 | 改用 `e.getMessage()` |
| 6 | 安全组端口未开放 | 腾讯云 8080 未开放 | 改用 HTTPS 隧道 |
