📋 PulseVita v2.0 需求文档

首页 & 三大功能页面全面优化

🎯 设计原则
简洁优先
减少视觉噪音，突出核心信息
数据驱动
用数据指导用户行为

成就系统、进度可视化
一致性
统一的视觉语言和交互模式
🏠 首页重新设计（7个功能）
FR-HOME-01: 顶部问候区 P0
显示个性化问候语、用户名、连续打卡天数
FR-HOME-02: 环境信息条 P0
左侧：天气图标 + 温度 | 右侧：节气图标 + 养生建议
FR-HOME-03: 核心数据卡片 P0
饮水进度、运动时长、今日心情的环形进度条展示
FR-HOME-04: 周概览卡片 P0
四宫格展示本周各维度健康数据：💧饮水 | 🏃运动 | 😊心情 | 🍎饮食
FR-HOME-05: 健康贴士卡片 P1
卡片式设计，支持翻转动画，每日自动更新
FR-HOME-06: 成就徽章区 P1
横向滚动展示已获得的成就徽章
FR-HOME-07: 快速操作区 P2
快速记录饮水/运动/心情，浮动操作按钮
🍽️ 饮食页面优化（5个功能）
FR-DIET-01: 饮水追踪卡片 P0
环形进度条 + 快速添加按钮：+100ml | +250ml | +500ml
FR-DIET-02: 营养摄入概览 P1
三个指标：蛋白质、碳水化合物、脂肪的进度条展示
FR-DIET-03: 餐次记录列表 P0
时间线设计，餐次图标区分：☀️早餐 | 🌤️午餐 | 🌙晚餐 | 🍪加餐
FR-DIET-04: 添加饮食记录 P0
浮动操作按钮，表单字段：餐次、食物名称、数量、单位
FR-DIET-05: 饮水历史图表 P2
柱状图展示每日饮水量，目标线标注
🏃 运动页面优化（6个功能）
FR-EX-01: 今日运动概览卡片 P0
双环进度条：运动时长（外圈）+ 步数（内圈）
FR-EX-02: 周运动趋势图 P0
柱状图展示每日运动时长，支持切换：时长/步数/卡路里
FR-EX-03: 运动类型统计 P1
三宫格展示主要运动类型，图标 + 名称 + 本周次数
FR-EX-04: 运动记录列表 P0
按时间倒序，每条记录：时间、类型、时长、卡路里
FR-EX-05: 添加运动记录 P0
浮动操作按钮，表单字段：运动类型、时长、距离
FR-EX-06: 运动目标设定 P2
每日运动时长/步数目标，每周运动次数目标
😌 心理页面优化（6个功能）
FR-MENTAL-01: 今日心情卡片 P0
5个表情图标：😫(1分) | 😔(2分) | 😐(3分) | 😊(4分) | 😄(5分)
FR-MENTAL-02: 心情趋势图 P0
折线图展示每日心情评分，Y轴：1-5分，X轴：周一到周日
FR-MENTAL-03: 呼吸练习卡片 P1
4-7-8呼吸练习引导，动画引导：吸气(4秒) → 屏息(7秒) → 呼气(8秒)
FR-MENTAL-04: 心情日记列表 P0
按时间倒序，每条记录：时间、表情、评分、文字描述
FR-MENTAL-05: 记录心情对话框 P0
对话框形式，心情评分：滑动条或表情选择
FR-MENTAL-06: 心情统计 P2
本周平均心情，心情最好/最差的一天
